ℹ️ About Khaali Spot

Khaali Spot is a real-time cafeteria seat availability tracker built for university campuses. Using computer vision and YOLOv8, it detects occupied and free seats from a camera feed and displays live availability to students.

Students no longer need to walk all the way to the cafeteria to find it full — they can check availability from their classroom in seconds.

Version: 1.0 MVP  |  Model: YOLOv8n  |  Built with: Python, Flask, OpenCV
